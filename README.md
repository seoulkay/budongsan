# 집값지도 - 아파트 실거래가 분석 서비스

국토교통부 실거래가 공공데이터를 기반으로 아파트 매매 추이를 분석하는 웹 서비스입니다.

## 시작하기

### 1. API 키 발급
1. [공공데이터포털](https://www.data.go.kr) 회원가입
2. "아파트매매 실거래 상세 자료" 검색 → 활용신청
3. 마이페이지 → 개발계정 → 일반 인증키(Decoding) 복사

### 2. 환경변수 설정
```bash
cp .env.local.example .env.local
# .env.local 파일 열어서 API 키 입력
```

### 3. 패키지 설치 및 실행
```bash
npm install
npm run dev
# http://localhost:3000 접속
```

## 배포 (Vercel)
```bash
npm install -g vercel
vercel
# 환경변수 MOLIT_API_KEY 설정 필요
```

## 기술 스택
- **프레임워크**: Next.js 14 (App Router)
- **스타일링**: Tailwind CSS
- **차트**: Recharts
- **데이터**: 국토교통부 실거래가 Open API

## 기능
- [x] 지역별 아파트 실거래가 조회 (서울/경기/부산)
- [x] 월별 평균가 추이 차트
- [x] 거래량 추이 차트
- [x] 거래량 상위 단지 TOP 10
- [x] 최근 실거래 내역 테이블
- [ ] 알림 기능 (이메일/카카오)
- [ ] 즐겨찾기 지역 저장
- [ ] CSV/엑셀 다운로드
- [ ] 전월세 데이터 추가
